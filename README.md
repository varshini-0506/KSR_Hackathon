# Vigil - Safety App

Vigil is a context-aware protection app that knows danger. Designed for women and elderly, it provides intelligent emergency response with adaptive geofencing and trusted circle alerts.

## Getting Started

This project is a starting point for a Flutter application.

### Prerequisites

- Flutter SDK (>=3.0.0)
- Dart SDK
- Android Studio / VS Code with Flutter extensions
- Android SDK (for Android development)
- Xcode (for iOS development on macOS)

### Installation

1. Install Flutter by following the [official installation guide](https://flutter.dev/docs/get-started/install)

2. Verify your installation:
   ```bash
   flutter doctor
   ```

3. Get the dependencies:
   ```bash
   flutter pub get
   ```

### Running the App

- Run the app:
  ```bash
  flutter run
  ```

- Run on a specific device:
  ```bash
  flutter run -d <device-id>
  ```

- List available devices:
  ```bash
  flutter devices
  ```

### Building the App

- Build APK for Android:
  ```bash
  flutter build apk
  ```

- Build iOS app:
  ```bash
  flutter build ios
  ```

## Features

- **Context-Aware Protection**: AI-powered risk detection using movement patterns, route familiarity, and companion presence
- **Dynamic Geofencing**: Adaptive safety zones that move with your trusted circle
- **Emergency Response**: Multi-tier escalation system with SOS button
- **Trusted Circle**: Social-aware geofencing with priority-based contacts
- **Privacy First**: On-device AI processing, no cloud tracking
- **Explainable AI**: Transparent risk monitoring with incident summaries
- **Guardian View**: Real-time location sharing for emergency responders
